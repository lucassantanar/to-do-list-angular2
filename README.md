## Funcionalidades

## Stack utilizada

**Front-end:** Angular 2+, SCSS



## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/lucassantanar/to-do-list-angular2-.git
```

Entre no diretório do projeto

```bash
  cd app-todo
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  ng s
```
